## mapcycle

* official cp + pl + koth

## server settings

```
 tf_clamp_airducks 1
 tf_use_fixed_weaponspreads 1
 tf_weapon_criticals 1
 sv_restrict_aspect_ratio_fov 0
```

## limitation

* all map sniper/spy/eng = 2
* allow connect from JP, TW, US, AS
* MaxPing 150
* AFK kick 180sec

## server command
* !mapvote (not votemap)

## server restart
* every 5am.
